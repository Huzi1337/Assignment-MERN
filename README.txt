//Running the application
1. Go to "MERN App"/Backend directory.
2. Run commands listed below:
    npm install
    npm start.
3. Go to "MERN App"/Frontend directory.
4. Run commands listed below:
    npm install
    npm run build
    npm run preview

//Testing the Backend
1. Go to "MERN App"/Backend directory
2. Run: "npm test".

//Testing the Frontend
1. Go to "MERN App"/Frontend directory
2.1. For unit and integration tests run:
     npm test.
2.2. For e2e test run: 
     npm run dev
     npx cypress run
